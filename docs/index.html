vue-tinybox

A slick, yet tiny lightbox gallery component for Vue.js

  • npm
  • github

What is it?

Tinybox is a simple lightbox gallery component for Vue.js.

While being not as customizable as some other components, Tinybox is dead simple and lightweight (less than 3.5 KB minzipped).

the size of Tinybox is controlled with Size Limit

Tinybox also doesn't depend on any third-party library, so that means pure JavaScript — with no mess.

Tinybox is great for mobile devices too: it supports swipe gestures and is responsive.

Install

Tinybox can be installed as an npm module:

npm install vue-tinybox
# or
yarn add vue-tinybox

...or served from CDN like jsDelivr:

<script
  src="https://cdn.jsdelivr.net/npm/vue-tinybox@0.3/dist/tinybox.umd.js"
>
</script>

Usage

Include the component like you normally would: via Vue.component(), Vue.use() or inside the components property of your component.

Then just pass the props needed: 

<Tinybox
  :images="images"
  :index="idx"
  loop
  @close="idx = null"
/>

For more info consult the README.
